Steps to Run the TTRCM Model Derivation & Cardiac Amyloid Cohort Validation.

1) Install Anaconda (Suggested as it comes with all required python packages & libraries essential for the model code.)
2) Decompress the package zip file
3) Open Jupyter Notebooks and navigate to the decompressed folder.
4) Launch Model Training ipynb file and goto "Cell" Menu in the menubar present and click Run All.
5) The model training performance can be tracked in the code blocks  titled "Model performance metrics".
6) Launch the Model Validation ipynb file and goto "Cell" Menu in the menubar present and click Run All.
7) Wait for the code blocks to execute and refer to the output files in the "Validation Outputs" generated once the code runs successfully for Validation Results.
